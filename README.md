# FRSGUI 
### Framework for Rapid and Simple development of General User Interface applications, based on SFML.

## About
FRSGUI is a lightweight GUI framework allowing for quick and easy creation on user interfaces.
This framework is a wrapper for SFML, it currently operates on [SFML](https://www.sfml-dev.org/) 3.0 and it has SFML already inside of it, so you do not need to add in manually for it to work. 

# How to use
## Installation
1. Git clone the repository from main branch
2. Copy the fonts folder to the directory of your executable file
3. Create a folder named FRSGUI in the directory of your executable file
4. Enter the FRSGUI folder from the git clone, copy resources folder and paste it into the newly created FRSGUI folder
5. Use provided CMakeLists.txt to compile FRSGUI alongside your program
